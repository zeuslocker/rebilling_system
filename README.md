# RebillingSystem

Auto Rebilling system

## Installation

Installation requires installed `asdf` on your machine.

1. Install ruby and all required dependencies
```bash
asdf install
```

2. Install all gems
```bash
bundle install
```

## Usage

Check Rspec tests for usage examples

Run
```bash
rspec
```
